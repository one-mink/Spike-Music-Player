# Spike-Music-Player
A simpe Rust music Player.

This App let you controll the volume and speed of a song.
You also can select between songs that ar stored in the ./music folder.

The music player uses the rodio library to controll songs and egui for the GUI.

## How to use
Put your music files into spike-music-player/music/

1. Downlaod files
2. Go to spike-music-player
3. Type:
